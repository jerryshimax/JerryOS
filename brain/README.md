# Brain Vault

An Obsidian-based knowledge vault with structured naming, auto-indexing, and frontmatter validation.

## Quick Start

1. Run `setup.sh` to create the vault at your configured path
2. Open Obsidian → "Open folder as vault" → select the path
3. Enable Dataview plugin when prompted

## Structure

- `Templates/` — File templates for each type
- `Indexes/` — Auto-maintained entity catalogs
- `Dashboard.md` — Master index with Dataview queries
- `Activity Log.md` — Auto-maintained by brain-guard.sh

## Naming Convention

`[Type] Entity - Description.md`

See [docs/BRAIN-GUIDE.md](../docs/BRAIN-GUIDE.md) for full details.
